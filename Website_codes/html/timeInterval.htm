The water quality monitoring system is controlled by Arduino and NODEMCU as a heart of the system. The main objective of water quality monitoring system is to observe real time physical and chemical property of the water. Whenever there is a change in the property of the water under the study it is detected by the Arduino microcontroller then send to NODENCU using serial communication the NODEMCU sent these data to a port using Https request the port store these data to MYSQL database using php script. The real time property of the water is accessed from the web address of the system.
